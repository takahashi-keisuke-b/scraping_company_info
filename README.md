# 会社概要クローラー（スクレイピングツール）

[![CI](https://github.com/takahashi-keisuke-b/scraping_company_info/actions/workflows/CI.yml/badge.svg)](https://github.com/takahashi-keisuke-b/scraping_company_info/actions/workflows/CI.yml)

ターゲットとなる企業サイトのURLを入力するだけで、サイト内を自律的に巡回（クローリング）し、会社概要ページから自動で企業情報を抽出するPythonアプリケーションです。

**URL**:
http://example.com

---
## 概要
企業サイトのどのようなページからでも会社概要のページを探り出し、精度よく情報を抽出することはできないかと思い、開発しました。

### 開発目的
- 営業リスト作成や市場調査に必要な情報収集を効率化するため
- Pythonを用いたスクレイピングの実践的な学習

### 機能

- **自動巡回（クローリング）**
  - 入力された企業URLのドメイン内を最大20ページまで自動探索。
  - スコアリングアルゴリズムにより、「会社概要」「アクセス」などの重要ページを優先的に判別して巡回。

- **3段階のデータ抽出（スクレイピング）**
  - ターゲットサイトの構造に合わせて、以下の3つの抽出ロジックを順に自動で試行し、網羅的にデータをキャッチ。
    1. **table（表）**: `<table>` タグで構築された、最も強固に構造化された表データからのマッピング。
    2. **dl（定義・説明リスト）**: 会社概要でよく使われる `<dl>`, `<dt>`, `<dd>` タグからの正確なマッピング。
    3. **flexible（フレキシブルブロック）**: `<div>` や `<li>` などの入れ子構造（最下層のツリー構造）から、項目名と値のペアを柔軟に検出してマトリックス形式に変換。

- **ソースの保持**
  - 取得した各データ（企業名、住所、電話番号など）が、「どのURLから抽出されたものか」というソース情報（URL）を個別に紐づけて管理。
